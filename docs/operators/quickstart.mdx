# Operator Quickstart

Welcome to CyberAi! This guide will get you from a fresh laptop to a fully operational CyberAi environment.

## Prerequisites

- Git installed and configured
- Node.js 18+ installed
- npm or yarn
- Terminal access

## Step 1: Clone the Repository

```bash
git clone https://github.com/SolanaRemix/CyberAi.git
cd CyberAi
```

## Step 2: Run Bootstrap

The bootstrap script will verify your environment, install dependencies, validate contracts, and build the site:

```bash
./tools/bootstrap/bootstrap.sh
```

## Step 3: Verify Installation

Run the audit script to ensure everything is set up correctly:

```bash
./tools/audit/audit.sh
```

All checks should pass with no critical issues.

## Step 4: Explore the Repository

### Contracts

View registered agents and repositories:

```bash
ls -la contracts/agents/
ls -la contracts/repos/
```

### Prompts

Review operator and agent prompts:

```bash
cat prompts/operators/security-audit.md
cat prompts/agents/copilot-core/bootstrap.md
```

### Site

Start the development server:

```bash
cd site
npm run dev
```

Visit http://localhost:4321 to see the CyberAi site.

## Step 5: Make Your First Change

### Add a New Contract

1. Create a contract file:

```bash
cp contracts/agents/example-agent.json contracts/agents/my-agent.json
```

2. Edit the contract with your agent details

3. Validate the contract:

```bash
ajv validate -s contracts/contract.schema.json -d "contracts/agents/my-agent.json" --strict=false
```

4. Commit and push:

```bash
git add contracts/agents/my-agent.json
git commit -m "Add my-agent contract"
git push
```

## Troubleshooting

### Bootstrap Fails

- Check Node.js version: `node --version` (must be 18+)
- Check npm version: `npm --version`
- Clear npm cache: `npm cache clean --force`
- Remove node_modules and reinstall: `rm -rf node_modules package-lock.json && npm install`

### Contract Validation Fails

- Check JSON syntax with `cat contracts/path/to/contract.json | jq`
- Validate against schema manually
- Review error messages carefully

### Site Build Fails

- Check for syntax errors in .astro files
- Verify all dependencies installed
- Clear build cache: `rm -rf site/dist site/.astro`
- Rebuild: `cd site && npm run build`
