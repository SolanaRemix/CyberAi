## ğŸ“– Core Concepts

### Contracts

CyberAi uses machine-readable contracts to define capabilities and requirements:

- **Agent Contracts** - Define AI services for security, analysis, and automation
- **Repository Contracts** - Specify repository capabilities and requirements
- **Runner Contracts** - Define execution environments for agents

Learn more in the [Contract Specification](#contracts).

### Agents

Intelligent agents provide automated services:

- **SmartContractAudit** - Security vulnerability scanning
- **SmartBrain** - AI-powered code analysis
- **GitAntivirus** - Malicious code detection
- **NodeAudit** - Dependency vulnerability scanning
- **ConflictsResolver** - Automated merge conflict resolution

See [Agent Documentation](./smartbrain/) for details.

### Terminal

The CyberAi Terminal is the command center for all operations:

- Execute security audits
- Deploy applications
- Manage configurations
- Monitor system status

Explore [Terminal Documentation](./terminal/) to learn more.
