# CyberAi Dashboard

Monitoring and management dashboard for the CyberAi ecosystem.

## Access

Visit: `https://cyberai.network/dashboard`

## Features

- Real-time metrics
- System status monitoring
- Workflow visualization
- Resource management
- Alert management

## Development

The dashboard is part of the main site. To develop:

```bash
cd site
npm install
npm run dev
```

Then visit `http://localhost:4321/dashboard`
