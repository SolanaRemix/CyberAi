# CyberAi Terminal

Command-line interface for CyberAi operations.

## Access

Visit: `https://terminal.cyberai.network` (or `https://cyberai.network/terminal`)

## Features

- Interactive command-line interface
- Full CyberAi command set
- Script automation
- Real-time execution
- History and autocomplete

## Development

```bash
npm install
npm run dev
```
